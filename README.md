# Spots

Spots is an interactive social media web application where users can add and remove photos, like photos of other users, and make a few minor adjustments to their profile.

## Description

A profile page where you can edit your profile's Avatar, Name, Occupation and Posts. After the header you are presented with the different posts of the profile with a title for these posts and a button to like or unlike said post.

## Tech Stack

- HTML
- CSS
- Responsive Design
